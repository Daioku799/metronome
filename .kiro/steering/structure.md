# Structure Steering

## Directory Layout
- `src/components/`: Reusable UI components.
- `src/hooks/`: Custom React hooks.
- `src/audio/`: Audio engine core (Main thread).
- `src/audio/worker/`: Web Worker source code.
- `src/types/`: Shared TypeScript interfaces and types.
- `src/utils/`: Generic utility functions.

## Naming Conventions
- Components: PascalCase (e.g., `MeterEditor.tsx`).
- Functions/Variables: camelCase.
- Files: camelCase or PascalCase based on content.
- Types/Interfaces: PascalCase.
